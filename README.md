# epw_viz

**Built with**
* [Dash] (https://plotly.com/dash/) - Framework for building the web app
* [Plotly Python] (https://plotly.com/python/) - Used to create the interactive plots 

**How to use this app**
```
git clone https://github.com/chrlng/epw_viz
cd epw_viz
```
Run this app locally by:
`python3 app.py`
